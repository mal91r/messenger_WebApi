Программа реализует весь функционал, включая дополнительный. 

Хочу обратить внимание на папку wwwroot. При генерации списка пользователей/сообщений в папке создается соответствующий json-файл. 
Если вы хотите заново провести инициализацию списков, то нужно удалить, либо исказить json-файлы - это не запрещено.
Для нормальной работы программы запрещено изменять содержимое папки wwwroot\data\. 
В случае внесения изменений программа не упадет, но могут возникнуть проблемы с генерацией новых данных.

Так же уточняющий момент - Post методы для инициализации списоков ничего не возвращают. 
Чтобы ознакомиться с полученными данными, нужно заново вызвать метод Get соответствующего контроллера.
